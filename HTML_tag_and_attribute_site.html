Building your first HTML doc!

When creating your first webpage, using whatever website sponsor, app or developer, you need to contain these specific tags.

  • The doctype html tag
  • The HTML tag
  • The head tag
  • The body tag

ON <DOCTYPE HTML>

Creating any website is hard enouh, starting the proccess though, is easy.
On your first line of code, you want to type "<!DOCTYPE HTML>". This special tag ensures that the code you use is translated by the most up-to-date, or the most recent version, of HTML. This allows for website consistancy, as well as provides access to ALL the other tags you will learn about/use. You only need declare this once, at the top line of your code.

ON <HTML>

The HTML tag essentially initializes your website. Without it, nothing you type would run right. It tels your computer to identify tags inside it, and create data, content and other stuff accordingly. However, unlike the doctype tag, and like most other tags, your html tag is paired. It is initiated with a <html> and closed with </html> all other tags go inside this tag, it forms your website.
